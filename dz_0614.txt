1. Создать новый репозиторий
2. Добавить файлы с занятия 14-го июня.
3. В файле index.php должна быть форма задающая значение переменной n.
4. В выводе необходимо 
	- с помощью циклов посчитать факториал n
	- Вывести фразу Hello word n раз
	- Увеличить n в 5 раз.
	
Для проверки указать ссылку на репозиторий, и ветку.
Проект успешно выполнен если после запуска, в браузере по адресу localhost:8080 после формы(п.3) и успешного ввода n выводится контент пункта 4.
